EMMAGINA BACKEND v2.0.0 - CMS CLEAN

Objetivo:
- Unificar el control de páginas en un solo módulo: Editor del Sitio.
- Evitar parches acumulados del Page Builder anterior.
- Reparar automáticamente documentos duplicados o incompletos de la página Inicio.

Qué validar después del deploy:
1. /api/health debe responder OK.
2. /api/pages/home debe responder JSON con key home, slug inicio y blocks.
3. En admin/editor-sitio.html debe cargar la lista de páginas.
4. Si aparece un error del Editor, usar el botón Reparar conexión una vez.
5. Probar Guardar página y Guardar bloque.

Comandos:
npm install
npm run check
git add .
git commit -m "Emmagina backend v2.0 CMS clean"
git push
