Emmagina Backend v2.2 - Editor del Sitio Final

Esta versión reemplaza el Page Builder anterior por un Editor del Sitio estable y centralizado.

Rutas públicas:
- GET /api/pages/home
- GET /api/editor-sitio/home
- GET /api/editor-sitio/_status

Rutas admin nuevas:
- GET    /api/admin/editor-sitio/pages
- POST   /api/admin/editor-sitio/pages
- GET    /api/admin/editor-sitio/pages/:pageId
- PATCH  /api/admin/editor-sitio/pages/:pageId
- DELETE /api/admin/editor-sitio/pages/:pageId
- POST   /api/admin/editor-sitio/pages/:pageId/blocks
- PATCH  /api/admin/editor-sitio/pages/:pageId/blocks/:blockId
- DELETE /api/admin/editor-sitio/pages/:pageId/blocks/:blockId
- PUT    /api/admin/editor-sitio/pages/:pageId/reorder
- POST   /api/admin/editor-sitio/_repair

Compatibilidad:
- /api/pages/home sigue funcionando.
- /api/admin/pages sigue funcionando, pero ahora apunta al mismo motor estable del Editor del Sitio.

Almacenamiento:
- Colección MongoDB: site_editor_pages
- No usa las colecciones antiguas pages ni site_pages.
