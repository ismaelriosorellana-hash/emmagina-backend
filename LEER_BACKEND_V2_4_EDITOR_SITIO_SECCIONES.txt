Emmagina Backend v2.4 - Editor del Sitio con secciones

Cambios principales:
- Editor del Sitio usa páginas con sections[] y blocks[] dentro de cada sección.
- Mantiene compatibilidad con blocks[] plano para versiones anteriores.
- Agrega bloques category_sidebar y category_grid.
- Permite agregar, editar, eliminar y reordenar secciones.
- Permite agregar varios bloques dentro de una misma sección.
- Repara automáticamente la Home migrando bloques antiguos a secciones.

Rutas nuevas:
POST /api/admin/editor-sitio/pages/:pageId/sections
PUT /api/admin/editor-sitio/pages/:pageId/sections/reorder
PATCH /api/admin/editor-sitio/pages/:pageId/sections/:sectionId
DELETE /api/admin/editor-sitio/pages/:pageId/sections/:sectionId
